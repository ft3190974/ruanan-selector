# 01 封面
开场致谢，自我介绍。今天向各位领导汇报软安科技为厦门税务局提供的软件供应链安全解决方案。AI时代到来，开源大模型、MCP、Skill等新型开源形态正快速进入政务信息化场景，我们的SCA产品——源兮，正是为此而生。

# 02 AI时代政务软件供应链安全新挑战
各位领导请看，当前政务软件供应链正面临三大全新挑战。第一，AI开源项目激增——大模型、MCP、Skill等新型开源形态缺乏有效安全管控手段。第二，二进制盲区——很多税务系统的软件是开发商交付的二进制制品，源码不可见，成分不透明。第三，合规压力——SBOM软件物料清单和许可证合规已经成为监管硬性要求。传统的SCA检测手段已经无法覆盖这些新场景。

# 03 源兮SCA产品概述
源兮SCA是我们面向AI时代打造的全场景软件成分分析平台。四大核心数据：20多种语言全量覆盖，包括PyTorch、TensorFlow等AI框架；100T以上的全球最大开源组件知识库；Java零误报率，C/C++低于10%误报；同时支持源码和二进制两种分析模式。一个平台，全栈透明。

# 04 核心能力一：AI时代新威胁检测
这是业界首个将开源大模型、MCP工具链、Skill插件纳入成分分析范围的SCA平台。可以识别Llama、Qwen、DeepSeek等主流大模型的供应链依赖，检测MCP服务端和Skill插件的开源组件与许可证，深入分析PyTorch、TensorFlow等框架的组成。关键是，这些AI组件与传统开源组件在同一个平台统一管控，不需要额外工具。

# 05 核心能力二：检测技术领先 + 二进制成分分析
我们的检测引擎有三个核心优势。源码分析引擎支持20多种语言，跨文件跨函数追踪，是真正的编译型检测而非简单的正则匹配。二进制分析引擎支持100多种格式，专门解决开发商交付二进制制品的场景——反编译、提取指纹、匹配知识库、输出风险报告，全流程自动化。背后是100T以上的全球最大开源知识库，覆盖3000多万个组件版本。

# 06 核心能力三：漏洞深度分析 + 快速修复
我们不只是告诉您有漏洞。从CVE检出开始，我们做CVSS评分、可达路径追踪、影响范围评估，过滤掉那些不可达的"噪声"漏洞。然后AI提供基于上下文的修复建议，包括版本升级路径和兼容性评估。修复后自动复扫验证，整个过程可追溯，满足审计要求。这是一个完整的漏洞闭环。

# 07 核心能力四：安全预警 + 资产快速盘点
主动防御是安全的关键。我们的系统实时对接全球CVE、NVD、CNVD等情报源，当新漏洞爆发时，组件级精准推送，只通知真正受影响的资产。同时，通过CVSS和业务影响双维度评分模型进行优先级排序。特别重要的是，0day漏洞爆发时，可以在分钟级定位到受影响资产范围，让安全团队先于攻击者行动。

# 08 核心能力五：MCP/API/定制化嵌入
再好的安全工具，如果不能融入现有的开发流程，就是摆设。我们提供MCP标准协议集成，可以直接对接税务现有的IDE、CI/CD流水线、代码仓库。200多个RESTful API端点覆盖扫描、查询、报表、策略管理的全流程。同时支持UI品牌化、扫描策略自定义、报告模板定制、审批流程适配等深度定制。目标是让开发人员在编码时刻就获得实时安全告警，真正实现安全左移。

# 09 厦门税务SCA落地方案
针对厦门税务的具体场景，我们设计了四阶段全流程嵌入方案。开发阶段，IDE插件加MCP集成，编码时实时成分分析。采购阶段，对上架的开发商交付物进行二进制SCA强制扫描，不满足安全要求不予上线。测试阶段，接入CI/CD流水线，不通过安全门禁不合并不发布。运维阶段，持续监控加资产动态盘点，新漏洞实时预警。四个阶段形成完整的安全闭环。
